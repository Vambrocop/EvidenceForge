# Review Types and Question Frameworks

Method anchors: PRISMA 2020 for reporting, Cochrane Handbook for intervention-review methods, JBI Manual for diverse review types, and CEE standards for environmental systematic reviews/maps.

## Review Types

**Systematic review**: answers a focused question using explicit search, screening, extraction, and appraisal methods.

**Scoping review**: maps the breadth of a field, concepts, methods, and evidence gaps; may not estimate pooled effects.

**Evidence map**: visual or tabular map of evidence volume by population, exposure, outcome, and design.

**Rapid review**: streamlined systematic review with documented shortcuts.

**Meta-analysis**: statistical synthesis of comparable effect sizes from primary studies.

**Umbrella review**: review of systematic reviews/meta-analyses, usually focused on evidence quality and consistency.

**Second-order meta-analysis**: statistical synthesis of meta-analytic estimates or reviews, requiring explicit handling of overlap and dependence.

## Question Frameworks

**PICO**:

- Population
- Intervention
- Comparator
- Outcome

**PECO**:

- Population
- Exposure
- Comparator
- Outcome

Best for environmental epidemiology, ecology, observational exposure research, and policy exposures.

**PICOS**:

- Population
- Intervention
- Comparator
- Outcome
- Study design

**SPIDER**:

- Sample
- Phenomenon of interest
- Design
- Evaluation
- Research type

Useful for qualitative or mixed-method evidence.

## Choosing a Review Type

Use systematic review when the question is focused and eligibility can be defined upfront.

Use scoping review when the field is broad, concepts vary, or the user needs a map before effect synthesis.

Use meta-analysis only when effect sizes are comparable enough to combine.

Use umbrella review when the evidence base already has multiple reviews.

Use second-order meta-analysis only when review-level estimates are comparable and overlap/dependence can be handled.

## Common Misclassification Risks

- Calling a broad scoping map a systematic review without critical appraisal.
- Calling an umbrella review a second-order meta-analysis when no statistical synthesis is planned.
- Calling a narrative summary a meta-analysis because it discusses effect direction.
- Running a meta-analysis before effect-size comparability has been established.
