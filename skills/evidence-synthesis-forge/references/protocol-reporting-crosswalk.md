# Protocol and Reporting Crosswalk

Use this reference to align a review with major evidence-synthesis guidance.

## PRISMA 2020

Use for reporting systematic reviews and meta-analyses.

Minimum implications:

- report why the review was needed;
- define eligibility criteria;
- report information sources and search strategy;
- provide selection process;
- report data collection process;
- define effect measures and synthesis methods;
- show study selection flow;
- report included studies and exclusions;
- make limitations and funding/conflicts visible.

## Cochrane Handbook

Use for intervention reviews and rigorous meta-analysis planning.

Minimum implications:

- start with scope and question;
- define inclusion criteria before screening;
- plan search and selection;
- collect data using explicit forms;
- choose effect measures before pooling;
- assess risk of bias;
- choose meta-analysis or non-meta-analytic synthesis based on data structure;
- consider missing results and certainty of evidence.

## JBI Manual

Use when the review type is broader than a conventional intervention review.

Particularly useful for:

- scoping reviews;
- umbrella reviews;
- qualitative evidence;
- mixed-methods reviews;
- etiology/risk reviews;
- economic evidence.

Minimum implication: select the review type first, then choose the methods.

## CEE Standards

Use for environmental and ecological evidence synthesis.

Minimum implications:

- use an a priori protocol;
- define question elements such as PICO/PECO;
- include grey literature and supplementary search when relevant;
- test search comprehensiveness where possible;
- use dual screening or justify deviations;
- document exclusion reasons;
- provide structured extraction data;
- appraise study validity;
- justify narrative synthesis versus meta-analysis;
- avoid vote-counting as an effect-synthesis method.
