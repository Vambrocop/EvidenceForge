# CEE Alignment for Environmental Evidence

Use this reference for environmental and ecological evidence syntheses.

## Protocol

CEE standards emphasize an a priori protocol that defines:

- question;
- search;
- screening;
- critical appraisal;
- data extraction;
- synthesis.

Treat protocol drift as a bias risk. If eligibility or appraisal criteria change, report and justify the change.

## Search

Plan:

- bibliographic databases;
- subject-specific databases;
- grey literature;
- organizational websites;
- citation chasing;
- expert/stakeholder input when relevant.

Record:

- exact search strings;
- search dates;
- database/platform settings;
- restrictions by language, date, geography, or publication type.

## Screening

Use clearly defined eligibility criteria. Pilot them where possible.

Prefer independent dual screening, or justify any sampling/streamlined approach.

Report:

- number of unique records;
- exclusions at each stage;
- full-text exclusion reasons;
- unavailable full texts;
- uncertain eligibility cases.

## Data Extraction and Critical Appraisal

Extract:

- population/species;
- exposure/intervention;
- comparator;
- outcome;
- study design;
- geography;
- scale;
- effect estimate and uncertainty;
- metadata needed for effect modifiers.

Appraise:

- internal validity;
- external validity;
- confounding;
- pseudoreplication;
- spatial autocorrelation;
- measurement reliability;
- sample size and sampling design.

## Synthesis

Choose synthesis based on study comparability.

Use meta-analysis only when an average effect is meaningful. If not, use narrative synthesis, evidence mapping, structured tables, or visual summaries.

Do not use simple vote-counting of positive/negative studies as an effect estimate.
