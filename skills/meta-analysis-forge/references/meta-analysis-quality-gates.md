# Meta-Analysis Quality Gates

Use these gates before recommending statistical pooling.

## Gate 1: Construct Compatibility

Check:

- Are outcomes measuring the same construct?
- Are intervention/exposure definitions comparable?
- Are time windows comparable?
- Are populations comparable enough for an average effect to mean something?

If no, prefer evidence map or narrative synthesis.

## Gate 2: Effect-Size Compatibility

Check:

- Are estimates on the same scale?
- Can metrics be transformed without losing the estimand?
- Are adjusted and unadjusted estimates mixed?
- Are coefficients from incompatible models being combined?

If no, separate families or do not pool.

## Gate 3: Dependence

Check:

- multiple outcomes per study;
- multiple time points;
- multiple subgroups;
- multiple model specifications;
- overlapping samples;
- shared control groups.

If yes, use multilevel models, robust variance estimation, pre-specified selection rules, or sensitivity analysis.

## Gate 4: Risk of Bias

Check:

- study design quality;
- missing data;
- confounding;
- selective reporting;
- measurement validity;
- publication bias.

The synthesis should show whether conclusions change when high-risk studies are removed or down-weighted.

## Gate 5: Interpretation

Report:

- pooled effect;
- uncertainty;
- heterogeneity;
- prediction interval when possible;
- certainty/quality caveat;
- whether the pooled effect is descriptive or causal.
