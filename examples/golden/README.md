# Golden Examples

Golden examples show small inputs and expected behaviors for EvidenceForge skills. They are not benchmark datasets and should not be treated as real evidence.

## Included Examples

- `meta-analysis-minimal/`: mock coding sheet for `meta-analysis-forge` and the minimal R workflow.
- `ml-screening-log/`: mock screening log for `meta-ml-screener`.

## How To Use

Use these examples to check whether an agent response follows the intended workflow:

- validates inputs before analysis;
- separates method decisions from statistical execution;
- preserves human judgment;
- names limits instead of overclaiming.
