# Demo Evidence Plan

## Input

```text
Topic:
Do green finance policies improve firm environmental performance?

Potential outputs:
systematic review, meta-analysis, umbrella review, ML-assisted screening.
```

## EvidenceForge Output Sketch

### Review Type

Hybrid evidence synthesis:

- systematic review of primary empirical studies;
- first-order meta-analysis if effect sizes are comparable;
- umbrella review if prior reviews exist;
- ML-assisted screening and classification.

### PECO

| Element | Definition |
|---|---|
| Population | firms, plants, or industries |
| Exposure | green finance policies, green credit, green bonds, climate finance regulations |
| Comparator | unexposed or less-exposed firms/plants |
| Outcomes | emissions, pollution intensity, green patents, environmental scores |

### Synthesis Plan

1. Map eligible studies by design and outcome.
2. Extract effect sizes separately by metric.
3. Use random-effects or multilevel models only when effects are comparable.
4. Use narrative synthesis for incompatible outcomes.
5. Check publication bias and small-study effects where study count permits.
6. Treat ML-discovered moderators as exploratory.
